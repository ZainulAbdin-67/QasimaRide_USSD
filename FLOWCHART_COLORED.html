Collections & Object Relationships

Component Type Purpose Example
StandManager Core Controller Manages all stands & users coordinateRides()
HashMap<String, RickshawStand> Collection Store multiple stands stands.get("CHARMINAR")
ArrayList<Driver> Collection Store drivers at each stand driversAtStand.add(driver)
Queue<Passenger> FIFO Collection Manage waiting passengers passengerQueue.poll()
User (Abstract) Base Class Common attributes for all users userId, name, phone
Driver Extends User Driver-specific properties vehicle, license, available
Passenger Extends User Passenger-specific properties source, destination, activeRide
